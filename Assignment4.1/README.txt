Instructions
Solve the problem and submit your solution before the deadline.


Develop the following programs with Android:

    A application, which has a user interface consisting of a text view and two buttons. Each time the first button is clicked the application should display the current date and time on the text view. When the second button is clicked, the content of the text should be removed.
    An application, which displays a blog user interface consisting of necessary labels and text fields for user name and comment and a submit button. After pressing submit button, the application must make sure that the user name and comment text fields are not empty and then display the current user entry and previous ones on the same window. Each user entry should be listed so that first comes a number, then the date and time of submitting entry, then user name and comment.
    A simple calculator application for doing basic arithmetic operations: +, -, x, / and %.
